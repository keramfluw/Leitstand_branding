PMO LEITSTAND – STREAMLIT APP

Diese App visualisiert einen PMO-Leitstand auf Basis einer Excel-Datei.

FUNKTIONEN
- Vier Zielebenen
- Statusvererbung Bottom-Up
- Verantwortliche
- Partnerziele filterbar

START
1. pip install -r requirements.txt
2. streamlit run app.py

DATEN
Datei: PMO_Leitstand_Zielstruktur_Template.xlsx

Nur Ziele der Ebene 4 erhalten einen manuellen Status.
Alle anderen werden automatisch berechnet.
